> this node models the person at the center of the ecosystem —
> not as a user profile but as a sovereign entity whose relations
> to systems, people, projects, and ideas evolve over time.
> any agent working in this ecosystem is working FOR this node.

## who

## what this node is

this is the zero-point of the ecosystem. every other node exists
in relation to this one. modeling this explicitly means the ecosystem
can reason about what you need, what you've already done, and what
you might want to do next — without you having to re-explain from
scratch each session.

## what this node is not

this is not surveillance. it does not track behavior, monitor
productivity, or optimize for engagement. it models relationships
so that agents can be helpful collaborators rather than blank slates.
you control what accretes here. anything that feels wrong gets pruned.
