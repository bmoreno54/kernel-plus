# kernel+

A sovereign ecology seed — structural commitments and operational protocols for building a personal knowledge ecosystem that grows from your friction.

## What This Is

Five foundational positions and seven trunk protocols, distilled through a veil of ignorance test from a living ecology. Domain-specific content stripped. What remains is what generalizes to any human building any kind of project ecosystem.

The kernel doesn't tell you what to build. It gives you structural logic — sovereignty, composition, provenance, ingestion, separation — that holds regardless of domain.

## Quick Start

1. Fork or clone this repo
2. Edit `alpha_entity.md` — claim it as yours
3. Read `kernel/` — five short documents, plain language
4. Read `trunk/` — seven operational protocols
5. Start working. The ecology grows from use.

Full bootstrap instructions in `CLAUDE.md`.

## Kernel+Extensions

The veil of ignorance is tunable. This repo is kernel+0 — maximum opacity, pure structure. When packaging for a specific audience, the veil relaxes in declared dimensions, producing bespoke seeds: kernel+music, kernel+gamedev, kernel+whatever. Same packaging procedure, different attenuation. See `CLAUDE.md` for details.
