## Integrated: Kernel Thickening — Vocabulary Migration

### Diachronic Provenance → thickened `kernel/provenance.md`

**Source:** Shadow mapper Run 14 (`analyses/vocabulary-migration-as-provenance-archaeology.md`)

**Domain-independent form:** Every ecology vocabulary begins in a domain-specific register and migrates toward domain-independence through use, following three stages: embodied encounter (tacit practice) → metaphorical articulation (domain-specific metaphor) → structural description (portable). The metaphorical stage is load-bearing — it carries embodied context the structural stage compresses away. The migration is compression, not loss. This is the diachronic form of the veil of ignorance.

**Evidence:** Vault archaeological trail — Zephyr (embodied encounter, 2024-2025), Fractil (metaphorical articulation, 2025-2026), Kernel (structural description, 2026). Seven specific commitment-vocabulary pairs traced across all three registers.

**Veil test:** PASS. Any collaborative system evolving conceptual vocabulary exhibits this pattern. Domain-independence emerges by abstracting FROM concrete encounter, not by starting abstract.

**Integration:** Added "Vocabulary Migration — The Diachronic Face of Provenance" section to provenance.md. Provenance now covers not just artifact origin chains but the migration path of conceptual vocabulary itself.
