# relation

a relation is intersubjective. both sides can describe it from their
own perspective. neither side contains the other.

a relation has a modality — the register it operates in. structural
relations (this produces that, this needs that) are one modality.
narrative, semantic, temporal, synesthetic relations are others.
no modality is lesser. contextualizing the modality is what makes
the relation traversable without flattening it.

a relation has a type, but the types are not closed. they accrete
as the graph encounters new kinds of connection. the vocabulary is
provisional. what is not provisional: both sides remain sovereign
after the relation is declared. if the relation breaks either node's
self-containment, it is not a relation — it is a dependency that
has been misnamed.

what crosses the boundary is the interface. some relations have
explicit interfaces (a function signature, a data shape). some
have none (a narrative resonance needs no contract). the interface,
when it exists, is the only place where two nodes share vocabulary.
